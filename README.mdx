# send-info-docs

The public knowledge base for the [Send](https://send.app) ecosystem. Built with Next.js 15 + MDX.

**Live at** [info.send.it](https://info.send.it)

## Quickstart

```bash
git clone https://github.com/<org>/send-info-docs.git
cd send-info-docs
npm install
npm run dev
```

## Documentation

| Section | What's Inside |
|---------|---------------|
| **Welcome** | Intro, mission, vision, team, contact |
| **Send App** | Overview, features, FAQs — Sendtags, rewards, referrals, savings |
| **Send Token** | $SEND tokenomics, bridge, upgrade |
| **Send Canton Wallet** | Pool Party Exchange, passkeys, multisig, Canton Coin |
| **CUSD Stablecoin** | Overview, reserves, FAQs |
| **Send Verified** | Verification requirements, access & rewards |
| **Finance** | Revenue, treasury, multisigs, emissions, funding rounds |
| **Miscellaneous** | Roadmap, metrics, contracts, brand assets, glossary |
| **Legal** | Terms, privacy, disclaimers, licenses |

## Project Structure

```
docs/          # MDX content
components/    # React components (Treasury, Revenue, PoolParty, SendVerified, etc.)
lib/           # Utilities, MDX config, data helpers
app/           # Next.js App Router
```

Custom components live in `components/` and are registered in `lib/md.js` for use in MDX files.

## Scripts

```bash
npm run dev     # local dev server
npm run build   # production build
npm run start   # serve production build
```
