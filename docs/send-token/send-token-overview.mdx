# Send Token Overview

$SEND is an ERC-20 utility token on the Base network. $SEND is used within the Send ecosystem to access features such as Sendtag registration, participation in reward distributions, and governance within the Send platform. It is not an investment product and does not represent equity, ownership, or any claim on revenue or profits.
