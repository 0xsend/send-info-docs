# Send Metrics

External dashboards and analytics for tracking Send ecosystem data in real-time.

## Dashboards

| Dashboard | Description |
|-----------|-------------|
| <a href="https://dune.com/send/send-metrics" className="multisig-token-link" target="_blank">Send Metrics (Dune)</a> | App activity, transfers, and user growth |
| <a href="https://dune.com/send/send-token-metrics" className="multisig-token-link" target="_blank">Send Token Metrics (Dune)</a> | Token supply, holders, and distribution data |

## On-Site Data

For treasury, revenue, and token data maintained by Send, see:

- [Treasury](/docs/finance/treasury) — Monthly holdings, total value, and historical snapshots
- [Revenue](/docs/finance/revenue) — Send App and stablecoin revenue with on-chain transaction links
- [Token Emissions](/docs/finance/token-emissions) — Circulating supply, inflation, vesting progress
- [Funding Rounds](/docs/finance/funding-rounds) — Contribution history and valuation data
- [CUSD Reserves Report](/docs/cusd-stablecoin/reserves-report) — Monthly CUSD attestations with Brale revenue reports
- [CUSD Ecosystem Fund](/docs/cusd-stablecoin/buybacks) — Fund revenue and allocation framework
