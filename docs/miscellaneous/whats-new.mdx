# What's New

A chronological log of notable updates to the Send ecosystem.

---

### April 2026

- **Pool Party Liquidity Pairs** — Added dedicated documentation for all active trading pairs on Pool Party Exchange
- **CUSD Ecosystem Fund** — Launched the Ecosystem Fund page detailing how CUSD reserve yield is deployed for Canton ecosystem growth
- **Campaign Terms** — Published campaign legal terms & conditions for promotional programs

- **Token Emissions Q1 2026** — End-of-quarter supply data: 346.3M circulating (34.63% of supply)

- **Treasury Dashboard** — Total treasury value card added with token price calculations
