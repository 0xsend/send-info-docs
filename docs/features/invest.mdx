# Invest

## Trading Fees

- **Send Fee:** 0.75% applied to every buy and sell to support Send product development.
- **Kyber Network Fee:** A separate liquidity provider fee collected by [Kyber Network](https://kyberswap.com/swap/base/eth-to-0xeab49138ba2ea6dd776220fe26b7b8e446638956) on each swap.

Your total cost equals the Send Fee plus the Kyber fee shown in the confirmation screen. Rate quotes display both so you can preview the combined impact before executing a trade.
