## How It Works

Send Connect injects a `window.canton` provider into web pages. dApp builders can detect it, request a user connection, and submit transactions. The extension handles passkey authentication, surfaces approval prompts, and routes signed transactions to Canton via the Send validator (`send-cantonwallet1`).

For end users it's a one-click install and a passkey sign-in. For dApp builders, integration is a handful of lines using the open-source `@sigilry/dapp` SDK.
